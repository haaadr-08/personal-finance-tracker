# 💰 FinTrack

FinTrack is a responsive personal finance web application that allows users to manage income and expenses, monitor their balance, and organize transactions with search, filtering, and sorting features.

## ✨ Features

- Add income and expense transactions
- Automatically calculate balance, total income, and total expenses
- Display total transactions, income entries, and expense entries
- Dynamic category selection based on transaction type
- Search transactions by title
- Filter transactions by income or expense
- Sort transactions by:
  - Newest Added
  - Oldest Added
  - Highest Amount
  - Lowest Amount
  - Latest Transaction Date
  - Earliest Transaction Date
- Delete transactions
- Store transactions using Local Storage
- Responsive design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Font Awesome
